2.8. if-Bedingung (if-Condition)

Eine abstrakte if-Bedingung sieht so aus:
if ausdruck
  programm
end
Das Programm zwischen dem Ausdruck (Expression) und end wird immer dann ausgeführt, wenn das Ergebnis des Ausdrucks nicht false und nicht nil ist.
Das Konstrukt für eine einfache if-Verzweigung in einem Ruby-Programm sieht wie folgendes Beispielprogramm aus:
a = 10

if a == 10 then    # a == 10 ist die Expression
  puts 'a ist 10'  # das Programm
end
Eine Expression kann man sehr gut im irb ausprobieren:
sw@debian:~/sandbox$ irb
irb(main):001:0> a = 10
=> 10
irb(main):002:0> a == 10
=> true
irb(main):003:0> exit

2.8.1. Kurzform

Eine häufig anzutreffende Kurzform einer if-Condition ist im folgenden Code zu finden:
a = 10

# Langversion
#
if a == 10 then
  puts 'a ist 10'           # Die Programmzeile ist von if und end eingeschlossen.
end

# Kurzversion
#
puts 'a ist 10' if a == 10  # das if folgt der Programmzeile

2.8.2. else

Man kann es sich schon denken, aber der Vollständigkeit halber auch hierzu noch ein kleines Beispiel:
a = 10

if a == 10 then
  puts 'a ist 10'
else
  puts 'a ist nicht 10'
end
