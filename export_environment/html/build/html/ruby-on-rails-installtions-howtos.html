1.3. Installations-Howtos

Ruby on Rails kann glücklicherweise auf sehr, sehr unterschiedlichen Betriebssystemen installiert werden. Die jeweiligen Installationswege sind dabei recht unterschiedlich. Zusätzlich muss man sich damit recht selten rumschlagen. Aus diesem Grund sind die Installationsbeschreibungen im Anhang des Buches und hier nur verlinkt.
