2.4. irb

irb steht für „Interactive-Ruby“ und ist eine Art Sandkasten in der man beliebig mit Ruby rumspielen kann. irb wird mit der Eingabe von irb auf der Shell gestartet und durch Eingabe von exit wieder beendet.
Ein Beispiel sagt mehr als tausend Worte:
sw@debian:~/sandbox$ irb
irb(main):001:0> puts 'Hello World!'
Hello World!
=> nil
irb(main):003:0> exit
sw@debian:~/sandbox$
In diesem Kapitel werden viele Beispiele innerhalb von irb erarbeitet. Sie werden auch später bei der Arbeit oft innerhalb von irb neue Sachen ausprobieren, weil es so herrlich schnell und unkompliziert ist.

2.4.1. Kommentare im irb

Kommentar innerhalb von irb machen logischweise selten richtig Sinn, da in irb erarbeiteter Code quasi Wegwerf-Code ist. Innerhalb dieses Buches werden aber immer mal wieder Kommentare zur Erklärung auch im irb benutzt. Diese können Sie problemlos mit kopieren und im irb einfügen oder auch weglassen.
