1.2. Rails 2.3.5 oder 3.0.x?

Das war beim Schreiben dieses Buches eine schwierige Entscheidung. Logischerweise war die Version 2.3.5 stabiler als eine 3.0.x. Trotzdem macht es für ein Buch, das in 2010 erscheint wenig Sinn sich mit einer eigentlich veralteten Rails Version zu beschäftigen. Ich werde auch nicht an allen Stellen beide Versionen mit einander vergleichen. Die Unterschiede sind zu gross! Wenn Sie ein Buch zu Rails 2.3.x suchen, dann müssen Sie noch mal kurz bei Amazon reinschauen. Dieses hier wird Ihnen nur mit 3.0.x helfen.

Wichtig

Eine Programmiersprache wie Ruby und ein Framework wie Rails verändern sich regelmässig. Schauen Sie deshalb bitte regelmässig auf http://www.ruby-auf-schienen.de nach. Dort werde ich aktuelle Infos und evt. Kapitelupdates veröffentlichen.
